# 鼠须管98五笔专版

如果你已经安装过了「鼠须管」，你只需要在终端中输入下面一条命令，就可获取这份配置到你的 Mac 上：


``````
sh -c "$(curl -fsSL https://gitee.com/wubi98/shell/raw/master/98wubi.sh)"
``````

# 删除鼠须管

如果你不想再使用鼠须管，终端输入如下一条命令将删除之：

``````
sudo rm -rf /Library/Input\ Methods/Squirrel.app
``````
